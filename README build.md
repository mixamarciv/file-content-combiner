**Как установать и собрать расширение**

# переходим на относительно актуальную версию nodejs:
nvm use 22.19.0

# устанавливаем пакеты необходимые для создания расширения:
npm install -g yo generator-code vsce
# если надо создать само расширение то запускаем: "yo code"

# устанавливаем зависимости:
npm install

# запускаем автоматическую сборку ts(если хотим вносить правки и отслеживать изменения):
npm run watch
# Далее для запуска проверки нажимаем F5

**Сборка расширения**
# Создаем .vsix файл:
vsce package

# Для установки расширения:
code --install-extension file-content-combiner-1.0.0.vsix
# после повторной установки нужно перезапустить расширение в самом vscode
